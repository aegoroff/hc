SHA256 Калькулятор Readme 

SHA256 калькулятор это простое консольное приложение позволяющее подсчитывать SHA256 хэш для:
- Переданной через командную строку строчки
- Файла
- Всех файлов в указанном каталоге, включая вложенные каталоги (ключ -r)

Кроме того есть:

- Восстанавление исходной строки по её хэшу методом грубой силы (перебор по конечному словарю)
- Поиск файла(ов) на диске по известному хэшу SHA256

Поддерживается:

- Исключающая фильтрация файлов по маске (ключ -e)
- Включающая фильтрация файлов по маске (ключ -i)
- Обработка вложенных каталогов (ключ -r)
- Файлы больше 4 Гб
- Вычисление хэша для части файла с заданием размера части и смещения относительно начала файла
- Проверка файла с использованием известного хэша SHA256 (валидация файла)
- Подсчет времени выполнения вычислений (ключ -t) для каждого файла
- Вывод суммы SHA256 в разном регистре

Использование

sha256 [OPTION] ...

Доступные опции:

-f [ --file ] <путь>              Полный путь к файлу для которого нужно посчитать SHA256
-d [ --dir ] <путь>	              Полный путь к каталогу для файлов которого нужно посчитать SHA256
-e [ --exclude ] <маска>          Маска (шаблон) файлов которые нужно исключать из обработки. 
                                  Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-i [ --include ] <маска>          Маска (шаблон) файлов которые нужно обрабатывать (остальные обрабатываться не будут).
                                  Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-s [ --string ] <строка>          Строка, для которой нужно подсчитать SHA256
-m [ --sha256 ] <SHA256 хэш>      SHA256 хэш по которому нужно либо проверить файл (задается с опцией -f) или 
                                  восстановить исходную строку (необходима опция -c)
-a [ --dict ] arg                 Словарь, по которому будет производиться восстановление строки по её хэшу
-n [ --min ] arg                  Минимальная длина строки для восстановления. По умолчанию 1
-x [ --max ] arg                  Максимальная длина строки для восстановления. По умолчанию равна длине словаря
-z [ --limit ] arg                Размер части файла в байтах для которой вычислять хэш.
                                  По умолчанию для всего файла
-q [ --offset ] arg               Стартовая позиция в файле начиная с которой вычислять хэш. По умолчанию ноль
-h [ --search ] <SHA256 хэш>      SHA256 хэш для поиска файла соответствующего ему
-o [ --save ] arg                 сохранить SHA256 хэши файлов в указанный файл.
-c [ --crack ]                    Восстановить указанный SHA256 хэш (найти захэшированную строку) заданный опцией sha256 (m)
-l [ --lower ]                    Выводить хэш в нижнем регистре
-r [ --recursively ]              Сканировать вложенные каталоги
-t [ --time ]                     Показывать время вычисления SHA256 (по умолчанию отключено)
-? [ --help ]                     Показать помощь


Примеры

Вычисление SHA256 хэша строки 123

sha256.exe -s 123


Вычисление SHA256 хэша файла

sha256.exe -f file.txt


Вычисление SHA256 хэша части файла (первый килобайт)

sha256.exe -f file.txt -z 1024


Вычисление SHA256 хэша части файла (один килобайт с пропуском первых 512 байт)

sha256.exe -f file.txt -z 1024 -q 512


Валидация файла по его хэшу

sha256.exe -f file.txt -m 0A3B10B4A34A250A87B47D538333F4B06589171C7DFEEE26FF84CC82BAC874FB


Вычисление SHA256 хэша всех файлов каталога c:\dir

sha256.exe -d c:\dir


Вычисление SHA256 хэша всех файлов каталога c:\dir а также всех его подкаталогов

sha256.exe -r -d c:\dir


Вычисление SHA256 хэша всех exe файлов каталога c:\dir

sha256.exe -d c:\dir -i *.exe


Вычисление SHA256 хэша всех файлов каталога c:\dir кроме файлов с расширением tmp

sha256.exe -d c:\dir -e *.tmp


Вычисление SHA256 хэша всех exe и dll файлов каталога c:\dir

sha256.exe -d c:\dir -i *.exe;*.dll


Вычисление SHA256 хэша всех exe файлов каталога c:\dir исключая те из них, которые начинаются с bad

sha256.exe -d c:\dir -i *.exe -e bad*


Поиск файла на диске C:\ по известному SHA256 хэшу

sha256.exe -d c:\ -r -h A665A45920422F9D417E4867EFDC4FB8A04A1F3FFF1FA07E998E86F7F7A27AE3


Восстановление строки по её хэшу используя словарь по умолчанию

sha256.exe -с -m A665A45920422F9D417E4867EFDC4FB8A04A1F3FFF1FA07E998E86F7F7A27AE3


Восстановление строки по её хэшу используя свой словарь

sha256.exe -с -m A665A45920422F9D417E4867EFDC4FB8A04A1F3FFF1FA07E998E86F7F7A27AE3 -a 0123456789


Восстановление строки по её хэшу используя свой словарь и определенные длины строки

sha256.exe -с -m A665A45920422F9D417E4867EFDC4FB8A04A1F3FFF1FA07E998E86F7F7A27AE3 -a 0123456789 -n 2 -x 6
