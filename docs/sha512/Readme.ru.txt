SHA512 Калькулятор Readme 

SHA512 калькулятор это простое консольное приложение позволяющее подсчитывать SHA512 хэш для:
- Переданной через командную строку строчки
- Файла
- Всех файлов в указанном каталоге, включая вложенные каталоги (ключ -r)

Кроме того есть:

- Восстанавление исходной строки по её хэшу методом грубой силы (перебор по конечному словарю)
- Поиск файла(ов) на диске по известному хэшу SHA512

Поддерживается:

- Исключающая фильтрация файлов по маске (ключ -e)
- Включающая фильтрация файлов по маске (ключ -i)
- Обработка вложенных каталогов (ключ -r)
- Файлы больше 4 Гб
- Вычисление хэша для части файла с заданием размера части и смещения относительно начала файла
- Проверка файла с использованием известного хэша SHA512 (валидация файла)
- Подсчет времени выполнения вычислений (ключ -t) для каждого файла
- Вывод суммы SHA512 в разном регистре

Использование

sha512 [OPTION] ...

Доступные опции:

-f [ --file ] <путь>              Полный путь к файлу для которого нужно посчитать SHA512
-d [ --dir ] <путь>	              Полный путь к каталогу для файлов которого нужно посчитать SHA512
-e [ --exclude ] <маска>          Маска (шаблон) файлов которые нужно исключать из обработки. 
                                  Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-i [ --include ] <маска>          Маска (шаблон) файлов которые нужно обрабатывать (остальные обрабатываться не будут).
                                  Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-s [ --string ] <строка>          Строка, для которой нужно подсчитать SHA512
-m [ --sha512 ] <SHA512 хэш>      SHA512 хэш по которому нужно либо проверить файл (задается с опцией -f) или 
                                  восстановить исходную строку (необходима опция -c)
-a [ --dict ] arg                 Словарь, по которому будет производиться восстановление строки по её хэшу
-n [ --min ] arg                  Минимальная длина строки для восстановления. По умолчанию 1
-x [ --max ] arg                  Максимальная длина строки для восстановления. По умолчанию равна длине словаря
-z [ --limit ] arg                Размер части файла в байтах для которой вычислять хэш.
                                  По умолчанию для всего файла
-q [ --offset ] arg               Стартовая позиция в файле начиная с которой вычислять хэш. По умолчанию ноль
-h [ --search ] <SHA512 хэш>      SHA512 хэш для поиска файла соответствующего ему
-o [ --save ] arg                 сохранить SHA512 хэши файлов в указанный файл.
-c [ --crack ]                    Восстановить указанный SHA512 хэш (найти захэшированную строку) заданный опцией sha512 (m)
-l [ --lower ]                    Выводить хэш в нижнем регистре
-r [ --recursively ]              Сканировать вложенные каталоги
-t [ --time ]                     Показывать время вычисления SHA512 (по умолчанию отключено)
-? [ --help ]                     Показать помощь


Примеры

Вычисление SHA512 хэша строки 123

sha512.exe -s 123


Вычисление SHA512 хэша файла

sha512.exe -f file.txt


Вычисление SHA512 хэша части файла (первый килобайт)

sha512.exe -f file.txt -z 1024


Вычисление SHA512 хэша части файла (один килобайт с пропуском первых 512 байт)

sha512.exe -f file.txt -z 1024 -q 512


Валидация файла по его хэшу

sha512.exe -f file.txt -m 6F6C7ED600C5E27023D63AF4F3943DDEF0309FE4CF2F6C4630985F06639FCDE93AB55EE9821D576C625A99AD62A0E3E9CC2396622B271BA8D94BC29866F46923


Вычисление SHA512 хэша всех файлов каталога c:\dir

sha512.exe -d c:\dir


Вычисление SHA512 хэша всех файлов каталога c:\dir а также всех его подкаталогов

sha512.exe -r -d c:\dir


Вычисление SHA512 хэша всех exe файлов каталога c:\dir

sha512.exe -d c:\dir -i *.exe


Вычисление SHA512 хэша всех файлов каталога c:\dir кроме файлов с расширением tmp

sha512.exe -d c:\dir -e *.tmp


Вычисление SHA512 хэша всех exe и dll файлов каталога c:\dir

sha512.exe -d c:\dir -i *.exe;*.dll


Вычисление SHA512 хэша всех exe файлов каталога c:\dir исключая те из них, которые начинаются с bad

sha512.exe -d c:\dir -i *.exe -e bad*


Поиск файла на диске C:\ по известному SHA512 хэшу

sha512.exe -d c:\ -r -h 3C9909AFEC25354D551DAE21590BB26E38D53F2173B8D3DC3EEE4C047E7AB1C1EB8B85103E3BE7BA613B31BB5C9C36214DC9F14A42FD7A2FDB84856BCA5C44C2


Восстановление строки по её хэшу используя словарь по умолчанию

sha512.exe -с -m 3C9909AFEC25354D551DAE21590BB26E38D53F2173B8D3DC3EEE4C047E7AB1C1EB8B85103E3BE7BA613B31BB5C9C36214DC9F14A42FD7A2FDB84856BCA5C44C2


Восстановление строки по её хэшу используя свой словарь

sha512.exe -с -m 3C9909AFEC25354D551DAE21590BB26E38D53F2173B8D3DC3EEE4C047E7AB1C1EB8B85103E3BE7BA613B31BB5C9C36214DC9F14A42FD7A2FDB84856BCA5C44C2 -a 0123456789


Восстановление строки по её хэшу используя свой словарь и определенные длины строки

sha512.exe -с -m 3C9909AFEC25354D551DAE21590BB26E38D53F2173B8D3DC3EEE4C047E7AB1C1EB8B85103E3BE7BA613B31BB5C9C36214DC9F14A42FD7A2FDB84856BCA5C44C2 -a 0123456789 -n 2 -x 6
