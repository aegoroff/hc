SHA384 Калькулятор Readme 

SHA384 калькулятор это простое консольное приложение позволяющее подсчитывать SHA384 хэш для:
- Переданной через командную строку строчки
- Файла
- Всех файлов в указанном каталоге, включая вложенные каталоги (ключ -r)

Кроме того есть:

- Восстанавление исходной строки по её хэшу методом грубой силы (перебор по конечному словарю)
- Поиск файла(ов) на диске по известному хэшу SHA384

Поддерживается:

- Исключающая фильтрация файлов по маске (ключ -e)
- Включающая фильтрация файлов по маске (ключ -i)
- Обработка вложенных каталогов (ключ -r)
- Файлы больше 4 Гб
- Проверка файла с использованием известного хэша SHA384 (валидация файла)
- Подсчет времени выполнения вычислений (ключ -t) для каждого файла
- Вывод суммы SHA384 в разном регистре

Использование

sha384 [OPTION] ...

Доступные опции:

-f [ --file ] <путь>              Полный путь к файлу для которого нужно посчитать SHA384
-d [ --dir ] <путь>	              Полный путь к каталогу для файлов которого нужно посчитать SHA384
-e [ --exclude ] <маска>          Маска (шаблон) файлов которые нужно исключать из обработки. 
                                  Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-i [ --include ] <маска>          Маска (шаблон) файлов которые нужно обрабатывать (остальные обрабатываться не будут).
                                  Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-s [ --string ] <строка>          Строка, для которой нужно подсчитать SHA384
-m [ --sha384 ] <SHA384 хэш>      SHA384 хэш по которому нужно либо проверить файл (задается с опцией -f) или 
                                  восстановить исходную строку (необходима опция -c)
-a [ --dict ] arg                 Словарь, по которому будет производиться восстановление строки по её хэшу
-n [ --min ] arg                  Минимальная длина строки для восстановления. По умолчанию 1
-x [ --max ] arg                  Максимальная длина строки для восстановления. По умолчанию равна длине словаря
-h [ --search ] <SHA384 хэш>      SHA384 хэш для поиска файла соответствующего ему
-o [ --save ] arg                 сохранить SHA384 хэши файлов в указанный файл.
-c [ --crack ]                    Восстановить указанный SHA384 хэш (найти захэшированную строку) заданный опцией sha384 (m)
-l [ --lower ]                    Выводить хэш в нижнем регистре
-r [ --recursively ]              Сканировать вложенные каталоги
-t [ --time ]                     Показывать время вычисления SHA384 (по умолчанию отключено)
-? [ --help ]                     Показать помощь


Примеры

Вычисление SHA384 хэша строки 123

sha384.exe -s 123


Вычисление SHA384 хэша файла

sha384.exe -f file.txt


Валидация файла по его хэшу

sha384.exe -f file.txt -m AFE0F32AFCA5A9A8422A82FAFB369C14342791EC780D8825465D3B8960A6EA6575EFF9DC5A7C8C563EC39E043E76CCC5


Вычисление SHA384 хэша всех файлов каталога c:\dir

sha384.exe -d c:\dir


Вычисление SHA384 хэша всех файлов каталога c:\dir а также всех его подкаталогов

sha384.exe -r -d c:\dir


Вычисление SHA384 хэша всех exe файлов каталога c:\dir

sha384.exe -d c:\dir -i *.exe


Вычисление SHA384 хэша всех файлов каталога c:\dir кроме файлов с расширением tmp

sha384.exe -d c:\dir -e *.tmp


Вычисление SHA384 хэша всех exe и dll файлов каталога c:\dir

sha384.exe -d c:\dir -i *.exe;*.dll


Вычисление SHA384 хэша всех exe файлов каталога c:\dir исключая те из них, которые начинаются с bad

sha384.exe -d c:\dir -i *.exe -e bad*


Поиск файла на диске C:\ по известному SHA384 хэшу

sha384.exe -d c:\ -r -h 9A0A82F0C0CF31470D7AFFEDE3406CC9AA8410671520B727044EDA15B4C25532A9B5CD8AAF9CEC4919D76255B6BFB00F


Восстановление строки по её хэшу используя словарь по умолчанию

sha384.exe -с -m 9A0A82F0C0CF31470D7AFFEDE3406CC9AA8410671520B727044EDA15B4C25532A9B5CD8AAF9CEC4919D76255B6BFB00F


Восстановление строки по её хэшу используя свой словарь

sha384.exe -с -m 9A0A82F0C0CF31470D7AFFEDE3406CC9AA8410671520B727044EDA15B4C25532A9B5CD8AAF9CEC4919D76255B6BFB00F -a 0123456789


Восстановление строки по её хэшу используя свой словарь и определенные длины строки

sha384.exe -с -m 9A0A82F0C0CF31470D7AFFEDE3406CC9AA8410671520B727044EDA15B4C25532A9B5CD8AAF9CEC4919D76255B6BFB00F -a 0123456789 -n 2 -x 6
