WHIRLPOOL Калькулятор Readme 

WHIRLPOOL калькулятор это простое консольное приложение позволяющее подсчитывать WHIRLPOOL хэш для:
- Переданной через командную строку строчки
- Файла
- Всех файлов в указанном каталоге, включая вложенные каталоги (ключ -r)

Кроме того есть:

- Восстанавление исходной строки по её хэшу методом грубой силы (перебор по конечному словарю)
- Поиск файла(ов) на диске по известному хэшу WHIRLPOOL

Поддерживается:

- Исключающая фильтрация файлов по маске (ключ -e)
- Включающая фильтрация файлов по маске (ключ -i)
- Обработка вложенных каталогов (ключ -r)
- Файлы больше 4 Гб
- Проверка файла с использованием известного хэша WHIRLPOOL (валидация файла)
- Подсчет времени выполнения вычислений (ключ -t) для каждого файла
- Вывод суммы WHIRLPOOL в разном регистре

Использование

whirlpool [OPTION] ...

Доступные опции:

-f [ --file ] <путь>                 Полный путь к файлу для которого нужно посчитать WHIRLPOOL
-d [ --dir ] <путь>	                 Полный путь к каталогу для файлов которого нужно посчитать WHIRLPOOL
-e [ --exclude ] <маска>             Маска (шаблон) файлов которые нужно исключать из обработки. 
                                     Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-i [ --include ] <маска>             Маска (шаблон) файлов которые нужно обрабатывать (остальные обрабатываться не будут).
                                     Можно указать несколько масок разделяя их точкой с запятой ";". Используется вместе с опцией -d (--dir)
-s [ --string ] <строка>             Строка, для которой нужно подсчитать WHIRLPOOL
-m [ --whirlpool ] <WHIRLPOOL хэш>      WHIRLPOOL хэш по которому нужно либо проверить файл (задается с опцией -f) или 
                                     восстановить исходную строку (необходима опция -c)
-a [ --dict ] arg                    Словарь, по которому будет производиться восстановление строки по её хэшу
-n [ --min ] arg                     Минимальная длина строки для восстановления. По умолчанию 1
-x [ --max ] arg                     Максимальная длина строки для восстановления. По умолчанию равна длине словаря
-h [ --search ] <WHIRLPOOL хэш>      WHIRLPOOL хэш для поиска файла соответствующего ему
-o [ --save ] arg                    сохранить WHIRLPOOL хэши файлов в указанный файл.
-c [ --crack ]                       Восстановить указанный WHIRLPOOL хэш (найти захэшированную строку) заданный опцией whirlpool (m)
-l [ --lower ]                       Выводить хэш в нижнем регистре
-r [ --recursively ]                 Сканировать вложенные каталоги
-t [ --time ]                        Показывать время вычисления WHIRLPOOL (по умолчанию отключено)
-? [ --help ]                        Показать помощь


Примеры

Вычисление WHIRLPOOL хэша строки 123

whirlpool.exe -s 123


Вычисление WHIRLPOOL хэша файла

whirlpool.exe -f file.txt


Валидация файла по его хэшу

whirlpool.exe -f file.txt -m BD801451D24470DF899173BFF3C04E875BE46C97D1529F84269C70C26C0F7D31D1AD21CBD985E7CFD7E1496B3BA5905789BC0790817DA26DC36D7ECA14B689D7


Вычисление WHIRLPOOL хэша всех файлов каталога c:\dir

whirlpool.exe -d c:\dir


Вычисление WHIRLPOOL хэша всех файлов каталога c:\dir а также всех его подкаталогов

whirlpool.exe -r -d c:\dir


Вычисление WHIRLPOOL хэша всех exe файлов каталога c:\dir

whirlpool.exe -d c:\dir -i *.exe


Вычисление WHIRLPOOL хэша всех файлов каталога c:\dir кроме файлов с расширением tmp

whirlpool.exe -d c:\dir -e *.tmp


Вычисление WHIRLPOOL хэша всех exe и dll файлов каталога c:\dir

whirlpool.exe -d c:\dir -i *.exe;*.dll


Вычисление WHIRLPOOL хэша всех exe файлов каталога c:\dir исключая те из них, которые начинаются с bad

whirlpool.exe -d c:\dir -i *.exe -e bad*


Поиск файла на диске C:\ по известному WHIRLPOOL хэшу

whirlpool.exe -d c:\ -r -h 344907E89B981CAF221D05F597EB57A6AF408F15F4DD7895BBD1B96A2938EC24A7DCF23ACB94ECE0B6D7B0640358BC56BDB448194B9305311AFF038A834A079F


Восстановление строки по её хэшу используя словарь по умолчанию

whirlpool.exe -с -m 344907E89B981CAF221D05F597EB57A6AF408F15F4DD7895BBD1B96A2938EC24A7DCF23ACB94ECE0B6D7B0640358BC56BDB448194B9305311AFF038A834A079F


Восстановление строки по её хэшу используя свой словарь

whirlpool.exe -с -m 344907E89B981CAF221D05F597EB57A6AF408F15F4DD7895BBD1B96A2938EC24A7DCF23ACB94ECE0B6D7B0640358BC56BDB448194B9305311AFF038A834A079F -a 0123456789


Восстановление строки по её хэшу используя свой словарь и определенные длины строки

whirlpool.exe -с -m 344907E89B981CAF221D05F597EB57A6AF408F15F4DD7895BBD1B96A2938EC24A7DCF23ACB94ECE0B6D7B0640358BC56BDB448194B9305311AFF038A834A079F -a 0123456789 -n 2 -x 6
